Visualization of Covid in US.

Covid is one of the most dangerous disease that we are seeing now. In this Diagram, you can see the current number of case on each state. There are multiple colors pointed out to multiple regions, Having thick red color indicates that there are more cases in that region in the recent time. Where as if the color is light red, it means there are very few cases in the past days. This diagram not only identifies cases by states, it also shows us how many cases are there in each county in a visual manner, so that we can understand properly.But, all the color and saturation principles are taken into consideration properly, because all the shades of red will look similar to naked eye when viewing the image.
